I'm Paul and I develop games in Java using a framework called LibGDX because I enjoy coding and I like to be in control of the process at every step. Yes, this is a little overwhelming at times, but hey... old habits die hard.

Mostly I make prototypes and enjoy bringing old classics to life again  with new features.

I'm the owner of Game Jam Argentina, a community dedicated to hosting game jams and local events in support of the indie movement.
