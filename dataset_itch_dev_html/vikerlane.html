Welcome to my humble Itch developer page.

I'm Pert, a professional web application developer by trade, currently living in South West England. My coder journey started in early 90s when my father showed me few simple commands for Basic on Commodore 64. It gave me amasing career, but I'll never forget how it felt to be 10 year old kid with endless possibilities to express one's creativity.  Thankfully to communities like Itch.io I've been able to get back to indie game development as hobby, doing mainly game jams and hopefully one day working my way up to releasing original game or two.

Currently using GameMaker Studio 2 and Aseprite as main tools.

I would love to hear your feedback, both positive and negative, so please leave all your comments for the games below.
