Kyatt/Impostor Cat Games

Kyatt/Impostor Cat Games www.impostorcat.com @kyatt7

Games by TJ Cordes (aka kyatt7) along with many talented musicians, voice actors, and pixel artists in the Western NY area. With exception of Fist's Elimination Tower and Go Morse Go! Arcade Edition, all of these games were made as part of game jams, so please understand that they're a bit rough around the edges. All of these games were made using HaxeFlixel.
