I grew up on video games so when I was a senior in high school I took a computer programming class to make my own games and software. I fell in love with logic, problem solving, and creating entire pieces of software from scratch. I created my first game in Visual Basic 6 titled ‘Neverworld’ during that class and never looked back. Creating video games is immensely fun and is my greatest passion.
