Memphis Game Developers MemphisGameDevelopers.org @MemphisGameDev

Memphis Game Developers (aka MGD) is an open group of developers and artists who share ideas and knowledge on game development as well as collaborate on projects and game jams. MGD is also the official Unity User Group for Memphis and the Greater Midsouth.


The Memphis Game Developers is a not-for-profit entity that relies on community support, so please consider supporting us through Patreon! https://www.patreon.com/MemphisGameDevs
